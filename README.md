## MVP(Minimum Viable Product)

1. 로그인 / 회원가입 / 마이페이지

-   user CRUD
-   JWT 발급과 이메일 인증을 통한 로그인

2. 병원, 약국 지도 페이지

-   카카오 맵 API를 활용한 병원 위치 조회

3. 사용자 마이페이지

-   진료 예약
-   병원 이용 리뷰 작성

4. 병원 상세 페이지

-   병원측과 환자측 간의 웹소켓을 이용한 실시간 채팅 상담
